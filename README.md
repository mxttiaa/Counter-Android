# Counter App (Versione 1.0)

Un'applicazione Android dal design moderno sviluppata come progetto pratico per imparare le basi della programmazione Android con Java e XML.

## Funzionalità Principali

* **Incremento e Decremento:** Pulsanti dedicati per aumentare o diminuire il valore.
* **Limiti di Sicurezza:** L'app impedisce di scendere sotto lo `0` e di superare il limite di `99999`, avvisando l'utente con un messaggio pop-up (Toast).
* **Reset Rapido:** Un pulsante per azzerare istantaneamente il contatore.
* **Easter Egg Visivo:** Ogni 10 numeri, lo sfondo dell'app cambia dinamicamente pescando da una palette di colori pastello.

## Design
I colori sono stati studiati per garantire un alto contrasto e una perfetta leggibilità.
